<h1 align="center">Crack Hash</h1>

Распределённа система для взлома MD5 хэшей

### Requirements

- Golang 1.24
- Docker
- Docker Compose

### Clone repository

```bash
git clone https://github.com/vitalioo/ris-crack-hash.git
```

### Running
Заполните sample.env необходимыми данными

#### Docker

```bash
docker compose up ---build
```
